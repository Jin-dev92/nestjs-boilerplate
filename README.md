## 인프라 구성

AWS ECS + Fargate 을 기본 골자로 구성.
현재 비용적 문제로 인해 ECS + Fargate 을 사용하지 않고, Docker-compose 를 사용하여 로컬 환경만 구성.

* Server Port: 4000
* Nginx Port: 80
* Postgres Port: 5432
* Prometheus Port: 9090
* Grafana Port: 3000
* Reserve proxy 설명 > nginx/nginx.conf conf 파일 참조.
* 컨테이너 내부 모니터링 기술 스택: Prometheus + Grafana
* 문서화: Swagger API

#### Hierarchy 정보
```
- /apis  -> nestjs
- / -> nginx > client > 추후 추가 예정
- /metrics -> prometheus
- /docs -> swagger
```

## How to run docker container?
Docker 콘테이너 실행
```bash
  * docker-compose 컨테이너 실행
  npm run docker-start-dev
  
  * docker-compose 종료 및 컨테이너 정리
  npm run docker-stop-dev
```
